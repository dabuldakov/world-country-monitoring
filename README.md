# world-country-monitoring
backend module

How to start:

1. install Gradle 
    ubuntu ``sudo apt install gradle -y``
2. reload project as Gradle, click on Gradle file, run command in terminal ``gradle build``
3. start postgres in docker file or with command ``docker-compose up``
4. start application

After you can fill database by scheduler controller in swagger
http://localhost:8080/swagger-ui/index.html
